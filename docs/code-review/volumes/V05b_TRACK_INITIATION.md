## 2. 文件职责与接口契约

### 2.1 职责

纯过程化 M/N 滑窗航迹起始器。维护长度为 N 的滑窗记录每帧检测情况，当滑窗内 ≥M 帧有检测且当前帧有检测时，触发多假设配对和共识评分。[S][P]

## 3. 逐语句块审查

### 3.1 滑窗维护（第 76—84 行）

**语句职责。** 追加当前帧点迹到滑窗，保持窗长 ≤N。[S]

**代码质量。** `state.window{end+1} = dets` 和 `state.has_det(end+1) = ...` 动态增长。N=8 时影响可忽略。[S]

**问题记录。** `V05b-P2-01`：`state.has_det(end+1)` 对 cell 数组使用索引增长，MATLAB 会先将 `has_det` 从标量数组扩展为 cell 数组。若 `params.tracker_N` 很大，首次扩展可能有性能开销。当前 N=8，无影响。[S]

### 3.2 M/N 条件检查（第 86—90 行）

**语句职责。** 滑窗内有检测的帧数 < M 或当前无检测时提前返回。[S]

**数学正确性。** `n_with_det = sum(state.has_det)` 统计滑窗内所有帧的检测情况。[M]

### 3.3 多假设配对（第 92—139 行）

**语句职责。** 三重循环：当前帧点迹 × 滑窗历史帧 × 历史帧内点迹。[S]

**性能问题。** 时间复杂度 `O(N_dets_current × N × N_dets_history)`。每帧最坏 `8 × 8 × N_dets^2`。在杂波密集场景（N_dets=10+），每帧约 6400 次 Haversine 距离计算。[S]

**问题记录。** `V05b-P2-02`：三重嵌套循环中 `sphere_utils_haversine_distance` 被调用最多 `N × N_dets_current × N_dets_history` 次。可提取为单次遍历的辅助函数避免重复计算。[S]

### 3.4 速度检验（第 107—113 行）

**语句职责。** Haversine 距离除以时间差估计速度，检验 [30, 600] m/s。[S]

**数学正确性。** `est_speed = dist / (dt_frames * dt_sec)`。速度范围 [30, 600] m/s 覆盖亚音速到高超音速。[M]

**问题记录。** `V05b-P2-03`：速度估计基于两点间的直线距离除以时间差，假设目标匀速直线运动。若目标在 `dt_frames * dt_sec` 内转弯或变速，估计速度失真。[M]

### 3.5 共识评分（第 115—130 行）

**语句职责。** 对其他含检测帧中的点迹，检查是否同时靠近 det1 和 det2 定义的轨迹线。[S]

**数学正确性。** `d1 < 80000 && d2 < 80000`：80 km 门限。若中间帧有点迹同时靠近 det1 和 det2 定义的直线轨迹，则认为该配对有共识。[M]

**问题记录。** `V05b-P1-01`：共识评分假设目标沿直线运动（det1→det2 连线方向）。若目标在滑窗期间转弯，中间帧的点迹可能偏离连线 80 km 门限，导致共识评分偏低，M/N 起始失败。在转弯场景中，M/N 起始器可能无法正确起始。[M]

**问题记录。** `V05b-P1-02`：共识评分的 80 km 门限是硬编码常数，未参数化。对 230 m/s、30s 采样的目标，80 km 对应约 5.8 帧的横向位移。若转弯半径小于 80 km（急转弯），共识评分可能完全失效。[M]

### 3.6 起始判定（第 141—146 行）

**语句职责。** `best_support >= 1` 即确认起始。[S]

**数学正确性。** 最低要求：至少 1 帧中间点迹提供共识支持。在 N=8、M=4 的滑窗中，这一门槛很低。[M]

**问题记录。** `V05b-P2-04`：`best_support >= 1` 门槛过低。单个离群点迹即可触发起始，可能导致虚警起始。建议 `best_support >= 2` 或与 M/N 条件联动。[S]

## 5. 文件级结论

- **数学可信度**：M/N 起始算法形式正确，但共识评分假设直线运动，转弯场景下可能失效。80 km 门限硬编码且过低。
- **代码质量**：三重嵌套循环性能差；`has_det` 类型转换隐患；共识评分门槛过低。
- **性能**：`O(N × N_dets^2)` 每帧，杂波密集场景显著。
- **测试充分性**：端到端冒烟通过，但转弯场景下的起始成功率未获独立验证。
- **剩余未验证项**：不同转弯半径下 M/N 起始的成功率；共识评分对 80 km 门限的敏感性。
