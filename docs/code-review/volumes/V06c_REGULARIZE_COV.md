## 3. 逐语句块审查

### 3.1 NaN/Inf 守卫（第 88—91 行）

**语句职责。** 若 `P` 含 NaN/Inf，返回单位阵。[S]

**代码质量。** 单位阵 `eye(n)` 是完全无先验信息的保守选择。但在极端情况下（如协方差爆炸），返回单位阵可能比返回 `diag([0.05^2, ...])` 更合理，因为单位阵的量级与 `P` 的原始量级无关。[S]

### 3.2 强制对称化（第 99 行）

**语句职责。** `P_sym = (P + P') / 2`。[S]

**数学正确性。** 对称化消除舍入误差导致的非对称分量。[M]

### 3.3 特征值分解（第 115 行）

**语句职责。** `[V, D] = eig(P_sym)`。[S]

**性能问题。** `eig` 对 4×4 矩阵开销小（`O(n³)`），但每帧每次融合都调用。在 MC 循环中（200-500 次 × 52 帧 × 4 算法），累计调用约 40,000-100,000 次。[S]

### 3.4 双阈值裁剪（第 126—140 行）

**语句职责。** `min_allowed = max(min_eig, 1e-6 * max(d))`，`d_clip = max(d, min_allowed)`。[S]

**数学正确性。** 双阈值策略合理：绝对阈值防止精确零特征值，相对阈值防止条件数过大。[M]

**问题记录。** `V06c-P2-01`：`max_d = max(d)` 可能为负（若所有特征值负）。此时 `min_allowed = max(1e-12, 负值) = 1e-12`。但如果 `max_d` 为负且 `min_eig` 未传入，`min_allowed` 仍为正，裁剪后所有特征值被提升到 `1e-12`，丢失了原始矩阵的方向信息。[M]

### 3.5 重构（第 153 行）

**语句职责。** `P_reg = V * diag(d_clip) * V'`。[S]

**数学正确性。** 重构保证对称正定。[M]
