## 2. 文件职责与接口契约

### 2.1 职责

空间配准核心模块：从 ADS-B 标校点估计两部雷达的系统偏差（距离和方位）。双阶段算法：LS 直接最小二乘初值 + EML（ECEF 空间最大似然）精化。[S][P]

### 2.2 输入输出

| 输入 | 类型 | 含义 |
|---|---|---|
| `r1_meas, r2_meas` | cell[n_frames] | 两部雷达量测序列 |
| `truth_points` | [N×2] | 标校点真实经纬度 |
| `radar*_lon, radar*_lat` | deg | 雷达位置 |
| `tx*_lon, tx*_lat` | deg | 发射站位置 |

| 输出 | 类型 | 含义 |
|---|---|---|
| `est` | 结构体 | `dr1, da1, dr2, da2` 四个偏差估计 |

## 3. 逐语句块审查

### 3.1 安全性检查（第 153—158 行）

**语句职责。** 有效标校点 < 3 时返回零偏置。[S]

**数学正确性。** 4 个未知数（dr1, da1, dr2, da2），至少需要 4 个独立方程。但 3 个点可提供 6 个方程（2 雷达 × 3 点），最小二乘可解。[M]

### 3.2 LS 估计（第 194—306 行）

**语句职责。** 逐帧计算 `量测 - 真实值`，取均值作为偏差估计。[S]

**数学正确性。**

```matlab
dr1_list = mean(m1.range_meas - true_rng1)
da1_list = mean(angle_wrap(m1.azimuth_meas - true_az1))
```

高斯噪声下样本均值是 MLE 和 BLUE。[M]

**方位角包裹**（第 251—255 行）：

```matlab
if daz1 > 180, daz1 -= 360; end
elseif daz1 < -180, daz1 += 360; end
```

标准角度归化到 [-180, 180]。[M]

**问题记录。** `V07b-P2-01`：`dr1_list(end+1)` 等动态数组增长。每帧最多 52 帧，影响可忽略。[S]

### 3.3 EML 优化点集构建（第 337—394 行）

**语句职责。** 下采样标校点（最多 50 个）构建优化点集。[S]

**数学正确性。** `opt_step = floor(n_cal / 50)`，均匀采样。[M]

**问题记录。** `V07b-P2-02`：下采样损失信息。若 52 帧中有漏检，实际可用点可能不足 50，下采样后更少。[S]

### 3.4 fmincon 优化（第 410—498 行）

**语句职责。** 以 LS 结果为初值，在 ECEF 空间优化偏差。[S]

**数学正确性。** 代价函数最小化 `Σ ||ECEF(corrected) - ECEF(truth)||²`。[M]

**边界设计**（第 448—463 行）：距离 ±50km，方位 ±10°。[S]

### 3.5 关键覆盖（第 514 行）

**语句职责。** `x_opt = x0` 无条件覆盖 fmincon 结果为 LS 初值。[S]

**问题记录。** `V07b-P0-01`：这是 CR-ISSUE-011 的核心。`fmincon` 执行并打印结果后，无条件覆盖为 LS 初值。"LS + EML 精化双阶段"的实际行为是"只使用 LS"。EML 增加了运行时间和 Optimization Toolbox 依赖，但对返回值无贡献。[S]
