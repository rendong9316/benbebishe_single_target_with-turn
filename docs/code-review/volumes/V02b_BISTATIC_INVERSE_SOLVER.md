## 3. 逐语句块审查

### 3.1 经典双基地反解初值（第 21—24 行）

**语句职责。** 假设 `Rg = r0 + r1`（地表大圆距离和），用双基地椭圆公式反解 `r1`。[S]

**数学正确性。**

```matlab
baseline = haversine_distance(tx, rx)           % 地表大圆基线
tx_az = azimuth(rx, tx)                          % Rx→Tx 方位角
phi = az - tx_az                                 % 方位角差
r1 = 0.5 * (Rg^2 - baseline^2) / (Rg - baseline * cos(phi))
```

这是经典双基地反解公式，假设 `Rg` 是两段地表大圆距离之和。[M]

**问题记录。** `V02b-P1-01`：`Rg` 是天波群距离（包含电离层路径），但反解公式假设 `Rg` 是地表大圆距离之和。天波斜距 `r = sqrt(D^2 + (2H)^2)` 大于地表距离 `D`，因此用天波 `Rg` 代入地表反解公式会产生系统性偏差。偏差量级：`H=300km`，`D=1000km` 时，`r ≈ sqrt(1000^2 + 600^2) ≈ 1166km`，偏差约 16.6%。[M]

### 3.2 钳位（第 27 行）

**语句职责。** `r1` 限制在 [1km, 5000km]。[S]

**数学正确性。** 合理范围：OTH-SWR 最小探测距离 1000km，最大 2000km。但此处钳位到 1km-5000km 覆盖了更广范围，防止迭代发散。[M]

### 3.3 迭代精化（第 30—39 行）

**语句职责。** 用天波模型预测 `Rg`，按比例修正 `r1`，直到收敛。[S]

**数学正确性。**

```matlab
tgt = destination_point(rx, r1, az)
Rg_pred = skywave_geometry('group_range', tx, rx, tgt)
err = Rg - Rg_pred
r1 = r1 * Rg / Rg_pred
```

定点迭代：`r1_new = r1 * Rg_true / Rg_pred(r1)`。收敛条件是 `Rg_pred ≈ Rg_true`。[M]

**问题记录。** `V02b-P2-01`：定点迭代 `r1 = r1 * Rg / Rg_pred` 是简单比例修正，不是牛顿法。收敛速度取决于 `Rg_pred` 对 `r1` 的敏感度。在远距离（`Rg≈2000km`）时，`Rg_pred` 对 `r1` 的变化率接近 1，收敛缓慢。30 次迭代收敛到 1m 精度可能不足。[M]

**问题记录。** `V02b-P2-02`：迭代中 `destination_point` 和 `skywave_geometry` 被调用 30 次。每次 `skywave_geometry` 内部约 5-10 次三角函数。在 MC 循环中，若每帧都调用反解（如偏差校正阶段），此开销不可忽视。[S]

## 5. 文件级结论

- **数学可信度**：经典双基地反解公式正确，但将天波群距离 `Rg` 代入地表大圆反解公式存在系统性偏差（~16%）。迭代精化使用定点法而非牛顿法，收敛速度可能不足。
- **代码质量**：文件简洁（43 行），注释清晰。
- **性能**：30 次迭代 × 每次 `skywave_geometry` 调用，在 MC 循环中是可感知的开销。
- **测试充分性**：端到端冒烟通过，但反解在不同 `Rg` 和 `az` 下的收敛性未获独立验证。
- **剩余未验证项**：反解误差对最终 RMSE 的影响；不同迭代次数对精度的影响。
