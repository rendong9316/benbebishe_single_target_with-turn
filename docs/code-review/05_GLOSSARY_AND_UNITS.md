# 术语、符号与单位基准

> 本页描述当前源码实际采用的数学契约。凡无法由源码唯一确认的物理含义均标记“待领域验证”，不得用术语名称替代单位推导。

## 1. 状态向量

基础 UKF、CV、CT 与 IMM 的统一状态顺序为

```text
x = [lon; lon_rate; lat; lat_rate]
```

| 索引 | 符号 | 当前含义 | 单位 |
|---:|---|---|---|
| 1 | `lon` | 经度 | deg |
| 2 | `lon_rate` | 经度变化率 | deg/s |
| 3 | `lat` | 纬度 | deg |
| 4 | `lat_rate` | 纬度变化率 | deg/s |

该顺序不是常见的 `[x;y;vx;vy]`，也不是 `[lon;lat;lon_rate;lat_rate]`。任何索引式读写都必须按此表核对。`tracker/inject_truth_velocity.m` 把 `x(4)` 写成经度且不返回结构体，已在问题台账登记。

状态向量混合角位置与角速度。若需要米制速度，源码通常使用近似换算

```text
v_north ≈ lat_rate × 111320
v_east  ≈ lon_rate × 111320 × cos(lat)
```

单位为 m/s。经度换算随纬度变化；固定使用 `111320` 而忽略 `cos(lat)` 会产生纬度相关误差。

## 2. 量测向量

当前 UKF 量测顺序为

```text
z = [group_range; azimuth; doppler_range_rate]
```

| 索引 | 符号 | 含义 | 单位 |
|---:|---|---|---|
| 1 | `Rg` | Tx—目标与目标—Rx 两段天波路径之和，即双基地群距离 | m |
| 2 | `az` | 从接收站指向目标的球面初始方位角 | deg，通常归一到 `[0,360)` |
| 3 | `vd` | 总群距离变化率 | m/s |

`vd` 在源码和文档中常称“多普勒速度”，但它不是 Hz 域多普勒频移。若需要频率，必须再结合载频或波长换算。方位是周期变量；直接线性求均值或相减在 0°/360° 边界可能错误。

## 3. 协方差矩阵单位

### 3.1 状态协方差 `P`

`P` 为 4×4，元素 `P(i,j)` 的单位是状态分量单位乘积，而不是统一的“度平方”：

- 位置—位置：deg²；
- 位置—速度：deg²/s；
- 速度—速度：deg²/s²。

因此直接用 `trace(P)` 把四个对角元相加虽在数值上合法，但混合了不同物理量纲。BC 互协方差中的迹收缩比和统一对角限制需要特别谨慎。

### 3.2 过程噪声 `Q`

`Q` 为 4×4 状态过程噪声协方差，元素单位应与对应 `P` 元素相同。当前基础实现采用经验对角矩阵和场景缩放，并非从连续白噪声加速度模型严格离散化得到。评价 `Q` 大小必须同时说明采样间隔和状态单位。

### 3.3 量测噪声 `R`

`R` 为 3×3，量测顺序与 `z` 一致：

- `R(1,1)`：m²；
- `R(2,2)`：deg²；
- `R(3,3)`：(m/s)²；
- 非对角元若存在，其单位是对应两种量测单位乘积。

当前主要配置为对角矩阵，即假设三类量测噪声互不相关。该假设是否符合天波传播物理尚待领域和数据验证。

### 3.4 创新协方差 `P_zz`

`P_zz` 为 3×3，其单位结构与 `R` 相同。`P_zz(1:2,1:2)` 是距离—方位二维子块，不含多普勒维。

## 4. 距离与角度

| 名称 | 定义 | 单位 |
|---|---|---|
| 地心角 `sigma` | 球心处两地理点夹角 | rad |
| 大圆弧长 | `R_e × sigma` | m 或 km，取决于 `R_e` |
| 弦长 `D` | `2 R_e sin(sigma/2)` | m |
| 单段天波斜距 `r` | `sqrt(D²+(2H)²)` | m |
| 群距离 `Rg` | `r_tx+r_rx` | m |
| Haversine 评估距离 | 以 `R=6371` 计算的大圆距离 | km |
| 方位角 | 局部北向顺时针角 | deg |

必须区分弦长、地表大圆弧长和天波斜距。三者在远距离场景下不能互换。函数名带 `_km_` 的评估辅助函数返回 km；`skywave_geometry` 使用 `R_e=6371000`，返回 m。

## 5. 马氏距离、NIS 与卡方门

平方马氏距离为

```text
d² = ν' S⁻¹ ν
```

其中 `ν` 是创新，`S` 是创新协方差。在模型成立且创新维数为 `k` 时，`d²` 理论上服从自由度 `k` 的卡方分布。

当前项目必须区分：

- NN/PDA 的距离—方位二维平方马氏距离：`k=2`；
- 完整距离—方位—多普勒三维创新 NIS：`k=3`；
- IMM 模型似然中使用的三维创新：应按 `k=3` 解释。

二维卡方的闭式 CDF 是 `F(x)=1-exp(-x/2)`。MATLAB R2023a 已核验：

| 阈值 | `chi2cdf(x,2)` |
|---:|---:|
| 4 | 0.8646647168 |
| 8 | 0.9816843611 |
| 12.5 | 0.9980695459 |
| 72 | 显示精度下约 1 |

因此 `pda_pd_gate=0.8647` 对应二维阈值 4。若关联器实际采用 `gate_sigma²×2` 且 `gate_sigma=6`，阈值是 72；二者不能同时解释为同一个波门概率。

## 6. 滤波与运动模型

- **CV（Constant Velocity）**：匀速模型。当前状态在经纬度角坐标中线性推进，并不是局部切平面米制 CV。
- **CT（Coordinated Turn）**：协调转弯模型。当前实现作用于经纬度变化率；其转弯率、纬度尺度和物理速度之间的严格对应关系需要单独验证。
- **UKF（Unscented Kalman Filter）**：用 Sigma 点传播近似非线性变换后的均值和协方差。
- **UT（Unscented Transform）**：无迹变换；权重和、Sigma 点尺度及负权重数值影响必须分别检查。
- **IMM（Interacting Multiple Model）**：通过 Markov 转移、模型混合、并行滤波、似然更新和矩组合融合 CV/CT 模型。组合协方差必须包含模型内协方差和模型均值离差外积。
- **NIS（Normalized Innovation Squared）**：标准化创新平方，即平方马氏距离；自由度等于实际创新维数。
- **NEES（Normalized Estimation Error Squared）**：状态估计误差相对状态协方差的二次型。当前评估模块尚未形成统一 NEES 报告。

## 7. 关联与航迹管理

- **NN**：最近邻，在波门内选择代价最小量测。
- **PDA**：概率数据关联，对单航迹的多个候选量测求边缘权重并形成等效创新。标准 PDA 还需处理漏检质量与协方差扩散项。
- **JPDA**：联合概率数据关联，应建立跨航迹互斥联合事件后再求边缘概率。当前 `jpda_multi.m` 更接近逐航迹独立门控和局部 PDA，不应无条件称为完整 JPDA。
- **M/N 起始**：最近 N 帧至少 M 次支持后确认航迹。真值辅助起始不属于该算法自身性能。
- **波门**：按预测创新统计限制候选量测集合。必须明确维数、平方/非平方距离、阈值和理论门概率。
- **杂波密度 `lambda`**：进入 PDA 归一化的空间密度；其量纲必须与所用量测空间体积一致。

## 8. 融合术语

- **SCC**：项目中的简单协方差组合，当前实现按独立估计的信息矩阵相加。它依赖独立性假设，不能简单等同于 `w=0.5` 的 CI。
- **BC（Bar-Shalom–Campo）**：已知或估计互协方差 `P12` 时的相关估计融合。当前 `P12` 递推为经验近似。
- **CI（Covariance Intersection）**：未知互相关时，以凸权重组合信息矩阵，目标是保持保守一致性。
- **FCI**：项目中的快速 CI，用协方差迹构造启发式权重；不是独立的普适标准定义，解释时必须引用当前实现。
- **互协方差 `P12`**：两个估计误差之间的交叉协方差，单位结构与 `P` 相同。
- **时间对齐**：把异步雷达航迹传播到共同时间基准。负时间步回推时是否加过程噪声取决于随机过程条件化问题，不能仅凭“回退”二字判断。

## 9. 证据和状态术语

证据码使用治理文档定义：`S` 源码、`M` 数学、`R` 运行、`B` 基准、`L` 文献、`H` 历史、`X` 冲突或未知。文件状态为 `NOT_STARTED / IN_PROGRESS / REVIEWED / VERIFIED / STALE_NEEDS_REREVIEW`。

“可运行”只说明特定命令和输入未异常退出；“REVIEWED”只说明覆盖完成；“VERIFIED”还要求关键结论达到规定证据等级。这三个概念不得互换。

## 10. 待确认术语

以下内容仍需文献或领域专家证据：

1. 文件注释中的“电离层双跳”究竟指每个 Tx/目标或目标/Rx 分段的两腿反射，还是整个双基地路径的两次电离层反射；
2. 固定虚高 `H=300 km` 对当前频段、时间和地理场景的适用范围；
3. 天波群距离率直接作为多普勒速度时的符号约定和载频换算接口；

5. 南阳子系统中的类型码、质量码和航迹报告字段，需要在 V10—V11 分卷中逐一建立数据字典。
